Bienvenue dans le jeu du Morpion

C'est quoi le morpion(le jeu hein pas l'insecte)?

  Le morpion est un jeu de réflexion se pratiquant à deux joueurs au tour par tour et dont le but est de créer le premier un alignement sur une grille.

Comment jouer au morpion ?

 Pour jouer une partie de morpion, il suffit de tracer sur une feuille blanche une grille de 3 cases sur 3 (selon les variantes, il est possible d’augmenter le nombre de cases). Le but du jeu est d’aligner avant son adversaire 3,4,5(selon les variantes) symboles identiques horizontalement, verticalement ou en diagonale. Chaque joueur a donc son propre symbole, généralement une croix pour l’un et un rond pour l’autre, ici nous utliserons un X et un Y. La partie se termine quand l’un des joueurs à aligner 3 symboles ou quand la grille est complétée sans vainqueur. Il y a alors égalité.

Comment gagner une partie de morpion ?

 Le premier joueur à aligner 3, 4 ou 5 symboles identiques gagne la partie. Attention, le joueur qui débute est toujours avantagé pour gagner. Pensez donc à alterner ! Sur ce site vous avez le choix entre jouer contre un ami ou contre l'ordinateur.

Maintenant à vous de jouer!

Pour cela, suivez les étapes qui suivent. Les scores s'affichent sous la grille après chaque partie.

1. Choisir la configuration de la grille.

2. Laisser coché pour jouer contre l'ordinateur ou décocher pour jouer à deux.

Jouer avec l'ordinateur.

3. Si vous êtes prêts, cliquez sur "Lancer le jeu".

4. A la fin d'une partie, une fenetre de dialogue s'affiche pour donner le resultat. Cliquer sur OK ou appuyer sur la touche entrée du clavier pour engristrer le score de la partie terminée.

5. cliquer à nouveau sur "Lancer le jeu" pour faire une nouvelle partie aves la meme configuration
